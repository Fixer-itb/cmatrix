# 矩阵求逆的几种方法

## 1. 高斯消元法(手算)

$[A|I_n]\stackrel{行变换}{\longrightarrow}[U|B]\stackrel{行变换}{\longrightarrow}[I_n|A^{-1}]$

## 2. LU分解

$A=LU\Rightarrow A^{-1}=U^{-1}L^{-1}$

其中L为下三角矩阵，U为上三角矩阵

## 3.SVD分解(奇异值分解)

$A=UWV^T\Rightarrow A^{-1}=VW^{-1}U^T$

其中矩阵U为正交矩阵，W为对角矩阵，V为正交矩阵

## 4. QR分解

 $A=QR\Rightarrow A^{-1}=R^{-1}Q^{-1}$

其中Q为正交矩阵，R为上三角矩阵
